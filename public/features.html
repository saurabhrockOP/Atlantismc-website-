What Minecraft version is supported?

Our network natively runs on 1.20.4, but you can connect with any client ranging from 1.16 to the latest release.

Are Bedrock players able to join?

Yes! Bedrock players on console, mobile, or Windows 10 can connect using the default Bedrock port (1932).
