gavel Server Rules

By playing on AtlantisMC, you automatically agree to follow all the rules listed below. Ignorance of the rules is not an excuse.

1. Chat Violations

No spamming, excessive cursing, bypassing the chat filter, advertising other servers, or directing toxicity towards other players.

2. Client Modifications (Hacking)

The use of X-ray, Aimbot, KillAura, Auto-Clickers, or any hacked client that gives an unfair advantage is strictly prohibited and results in a permanent ban.

3. Exploits & Glitches

Exploiting server bugs, duplication glitches, or block glitching is forbidden. Please report bugs on the forums or discord for a bounty reward.

4. Griefing & Scamming

Griefing claimed land or scamming players in real-money transactions (IRL trading) will lead to a severe network ban.

terminal Essential Commands

Command Description Aliases / Notes

/sethome [name] Sets a personal teleport location. Max homes based on Rank
/withdraw [amount] Withdraw hearts in Lifesteal. Be careful!
/ah Opens the Global Auction House. Safe trading menu
